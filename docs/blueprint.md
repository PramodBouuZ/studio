# **App Name**: BANTConfirm Marketplace

## Core Features:

- Hero Banner Management: Admin can upload and manage 5 hero banners with promotional content.
- Scrolling Benefits Text: Display a scrolling text highlighting platform benefits.
- AI-Powered Inquiry Refinement: Use AI to suggest relevant keywords or clarify requirements when a customer submits an inquiry.
- Product Catalog with Search and Filters: A searchable product catalog with filtering by price and product type, managed by admin.
- Vendor Showcase Carousel: Display up to 5 scrolling vendor logos, manageable via vendor panels.
- Customer Feedback Section: Showcase up to 4 customer testimonials and feedback.
- Admin Lead Management with AI insights: Verify leads, assign them to vendors, download leads, and view/edit user details with AI-generated suggestions to improve lead qualification.

## Style Guidelines:

- Primary color: Royal Blue (#4169E1), symbolizing trust and technology.
- Background color: Light gray (#F0F8FF), for a clean and modern interface.
- Accent color: Golden Yellow (#FFD700), emphasizing 'Confirm' and highlighting key actions.
- Font: 'Inter', a grotesque-style sans-serif font, for body and headline, lending a neutral, machined look.
- Use clear and professional icons for product categories and features.
- Ensure a clean and responsive layout, optimized for both desktop and mobile devices.
- Use subtle transitions and animations to enhance user experience, such as loading indicators and hover effects.
- Build with animation styles and user-friendly design principles.
- Implement notification emails when users create accounts or post inquiries, and when new inquiries are posted on the platform.
