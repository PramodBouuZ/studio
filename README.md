# BANTConfirm Marketplace

This is a Next.js application for BANTConfirm Marketplace, a platform designed to connect customers with the right vendors.

## Core Features

- **Hero Banner Management**: A dynamic hero section showcasing promotional content.
- **Scrolling Benefits Text**: A marquee highlighting the key benefits of the platform.
- **AI-Powered Inquiry Refinement**: Customers can submit inquiries which are then refined by AI to provide better vendor matches.
- **Product Catalog**: A searchable and filterable catalog of products and services.
- **Vendor Showcase**: A carousel of featured vendor logos.
- **Customer Feedback**: A section to display testimonials from satisfied customers.
- **Admin Lead Management**: A comprehensive dashboard for admins to manage leads, with AI-powered insights to improve lead qualification.

## Tech Stack

- **Framework**: Next.js (App Router)
- **Styling**: Tailwind CSS
- **UI Components**: shadcn/ui
- **AI**: Google Gemini via Genkit
- **Icons**: Lucide React
- **Language**: TypeScript

## Getting Started

To get started with the development server:

```bash
npm run dev
```

This will start the Next.js application on `http://localhost:9002`.
